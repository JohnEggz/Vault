# 3.4. Obliczyć f'(0), gdzie
$$
f(x) = \begin{cases}
x(x+1)^{2} & \text{dla }x\leq 0 \\
0 & \text{dla } x> 0
\end{cases}
$$
### Rozwiązanie zadania 3.4 - Obliczenie $f'(0)$

Aby sprawdzić istnienie pochodnej w punkcie sklejenia $x=0$, obliczamy pochodne jednostronne.

**1. Pochodna lewostronna ($f'_{-}(0)$)**
Dla $x \leq 0$ funkcja dana jest wzorem $f(x) = x(x+1)^{2}$.
Przekształcamy wzór, aby łatwiej policzyć pochodną:
$$f(x) = x(x^2 + 2x + 1) = x^3 + 2x^2 + x$$
Obliczamy pochodną:
$$f'(x) = 3x^2 + 4x + 1$$
Podstawiamy $x=0$:
$$f'_{-}(0) = 3(0)^2 + 4(0) + 1 = 1$$

**2. Pochodna prawostronna ($f'_{+}(0)$)**
Dla $x > 0$ funkcja jest stała: $f(x) = 0$.
Pochodna funkcji stałej wynosi 0:
$$f'(x) = 0$$
Zatem:
$$f'_{+}(0) = 0$$

**Wniosek:**
Porównujemy pochodne jednostronne:
$$f'_{-}(0) = 1 \quad \text{oraz} \quad f'_{+}(0) = 0$$
Ponieważ $f'_{-}(0) \neq f'_{+}(0)$, pochodna funkcji w punkcie $x=0$ **nie istnieje**.

W punkcie $(0,0)$ wykres funkcji posiada "ostrze" (punkt kątowy).
