# 3.5. Zbadać różniczkowalność funkcji f oraz sprawdzić, czy f należy do C^1 (R), gdzie:
5.a)
$$
f(x) = \begin{cases}
x\sin\frac{1}{x} & \text{dla } x\neq 0 \\
0 & \text{dla } x = 0
\end{cases}
$$
5.b)
$$
f(x) = \begin{cases}
x^{2}\arctan \frac{1}{x} & \text{dla } x \neq 0 \\
0 & \text{dla } x = 0 \\
\end{cases}
$$
### Rozwiązanie zadania 3.5 - Badanie różniczkowalności i klasy $C^1$

Aby funkcja należała do klasy $C^1(\mathbb{R})$, musi być różniczkowalna w każdym punkcie dziedziny, a jej pochodna $f'(x)$ musi być funkcją ciągłą.

---

#### 5.a) $$f(x) = \begin{cases} x\sin\frac{1}{x} & \text{dla } x\neq 0 \\ 0 & \text{dla } x = 0 \end{cases}$$

**1. Badanie różniczkowalności w $x=0$**
Obliczamy pochodną z definicji (granica ilorazu różnicowego):
$$f'(0) = \lim_{h \to 0} \frac{f(0+h) - f(0)}{h} = \lim_{h \to 0} \frac{h\sin\frac{1}{h} - 0}{h} = \lim_{h \to 0} \sin\frac{1}{h}$$
Granica ta **nie istnieje** (funkcja oscyluje między $-1$ a $1$ w nieskończoność).
Zatem funkcja **nie jest różniczkowalna** w punkcie $x=0$.

**2. Sprawdzenie klasy $C^1$**
Skoro funkcja nie ma pochodnej w punkcie $x=0$, to $f \notin C^1(\mathbb{R})$.

**Wniosek:** Funkcja nie jest różniczkowalna w zerze i nie należy do klasy $C^1(\mathbb{R})$.

---

#### 5.b) $$f(x) = \begin{cases} x^{2}\arctan \frac{1}{x} & \text{dla } x \neq 0 \\ 0 & \text{dla } x = 0 \end{cases}$$

**1. Obliczenie pochodnej dla $x \neq 0$**
Korzystamy ze wzorów na pochodną iloczynu i funkcji złożonej:
$$f'(x) = (x^2)' \arctan\frac{1}{x} + x^2 \left(\arctan\frac{1}{x}\right)'$$
$$f'(x) = 2x \arctan\frac{1}{x} + x^2 \cdot \frac{1}{1+(\frac{1}{x})^2} \cdot \left(-\frac{1}{x^2}\right)$$
Po uproszczeniu drugiego składnika ($\frac{x^2}{1+\frac{1}{x^2}} \cdot \frac{-1}{x^2} = \frac{-1}{1+\frac{1}{x^2}} = \frac{-x^2}{x^2+1}$):
$$f'(x) = 2x \arctan\frac{1}{x} - \frac{x^2}{x^2+1}$$

**2. Badanie różniczkowalności w $x=0$**
Liczymy pochodną z definicji:
$$f'(0) = \lim_{h \to 0} \frac{h^2 \arctan\frac{1}{h} - 0}{h} = \lim_{h \to 0} h \arctan\frac{1}{h}$$
Ponieważ $\arctan$ jest ograniczony ($-\frac{\pi}{2} < \arctan t < \frac{\pi}{2}$), a $h \to 0$, to granica wynosi $0$.
Zatem **funkcja jest różniczkowalna w zerze** i $f'(0) = 0$.

Wzór pochodnej dla całej dziedziny:
$$f'(x) = \begin{cases} 2x \arctan\frac{1}{x} - \frac{x^2}{x^2+1} & \text{dla } x \neq 0 \\ 0 & \text{dla } x = 0 \end{cases}$$

**3. Badanie ciągłości pochodnej (sprawdzenie warunku $C^1$)**
Aby $f \in C^1(\mathbb{R})$, granica pochodnej przy $x \to 0$ musi równać się wartości pochodnej w zerze ($f'(0)=0$).
$$\lim_{x \to 0} f'(x) = \lim_{x \to 0} \left( 2x \arctan\frac{1}{x} - \frac{x^2}{x^2+1} \right)$$
Analizujemy składniki:
* $\lim_{x \to 0} 2x \arctan\frac{1}{x} = 0$ (iloczyn zera i funkcji ograniczonej).
* $\lim_{x \to 0} \frac{x^2}{x^2+1} = \frac{0}{0+1} = 0$.

Zatem $\lim_{x \to 0} f'(x) = 0$.
Ponieważ $\lim_{x \to 0} f'(x) = f'(0)$, pochodna jest ciągła w zerze (w pozostałych punktach jest ciągła jako suma funkcji elementarnych).

**Wniosek:** Funkcja jest różniczkowalna w całej dziedzinie i **należy do klasy $C^1(\mathbb{R})$**.
