# 3.3. Obliczyć f'(2), gdzie
$$
f(x) = \begin{cases}
-x^{2}+x    & \text{dla } x\leq 2 \\
x^{2} -7x+8 & \text{dla } x>2
\end{cases}
$$

# Rozwiązanie zadania 3.3 - Obliczenie $f'(2)$

Ponieważ punkt $x=2$ jest punktem "sklejenia" dwóch różnych wzorów funkcji, aby obliczyć pochodną w tym punkcie, należy policzyć **pochodne jednostronne** i sprawdzić, czy są równe.

Wstępnie sprawdzamy ciągłość w $x=2$ (warunek konieczny różniczkowalności):
* $f(2) = -2^2 + 2 = -2$
* $\lim_{x\to 2^+} (x^2 - 7x + 8) = 4 - 14 + 8 = -2$
Funkcja jest ciągła, więc możemy badać pochodną.

**1. Pochodna lewostronna ($f'_{-}(2)$)**
Dla $x \le 2$ funkcja określona jest wzorem $f(x) = -x^2 + x$.
Liczymy pochodną tego wyrażenia i podstawiamy $x=2$:
$$f'(x) = (-x^2 + x)' = -2x + 1$$
$$f'_{-}(2) = -2(2) + 1 = -3$$

**2. Pochodna prawostronna ($f'_{+}(2)$)**
Dla $x > 2$ funkcja określona jest wzorem $f(x) = x^2 - 7x + 8$.
Liczymy pochodną tego wyrażenia i podstawiamy $x=2$:
$$f'(x) = (x^2 - 7x + 8)' = 2x - 7$$
$$f'_{+}(2) = 2(2) - 7 = 4 - 7 = -3$$

**Wniosek:**
Ponieważ pochodna lewostronna jest równa pochodnej prawostronnej:
$$f'_{-}(2) = f'_{+}(2) = -3$$

Zatem pochodna funkcji w punkcie $x=2$ istnieje i wynosi:
$$f'(2) = -3$$
