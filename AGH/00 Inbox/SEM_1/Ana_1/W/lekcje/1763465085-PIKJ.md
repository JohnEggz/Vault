# Twierdzenie o indukcji matematycznej

1. Udowodnienie prawdziwości twierdzenia dla pewnej liczby naturalnej $n$.
2. Założenie, że twierdzenie jest prawdziwe dla liczby naturalnej $k$, takiej że $k \ge n$.
3. Udowodnienie prawdziwości twierdzenia dla $k + 1$.

## Przykład

Mamy udowodnić, że wzór jest prawdziwy:
$$1 + 2 + 3 + \dots + n = \frac{n(n+1)}{2}$$

### 1. Sprawdzamy, czy równość jest prawdziwa dla $n = 1$
$$1 = 1$$
$$L = P$$

### 2. Zakładamy, że równanie jest prawdziwe dla $k \ge 1$
$$1 + 2 + 3 + \dots + k = \frac{k(k+1)}{2}$$

### 3. Wykorzystując założenie z punktu drugiego, udowadniamy prawdziwość równania dla $k + 1$
$$1 + 2 + 3 + \dots + k + (k+1) = \frac{(k+1)((k+1)+1)}{2}$$

Dowód:
$$k^2 + k + 2k + 2 = k^2 + 2k + k + 2$$
$$0 = 0$$
$$L = P$$
